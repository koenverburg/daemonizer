# Daemonizer CLI (WIP)

Intro

## Run Locally

Clone the project

```bash
  git clone https://github.com/koenverburg/daemonizer
```

Go to the project directory

```bash
  cd daemonizer
```

Install dependencies

```bash
  go install
```

Create a binary

```bash
  go build
```
